# Portfolio Landing Page

Одностраничный сайт-портфолио, который демонстрирует навыки в веб-дизайне и фронтенд-разработке: визуальная система, адаптивная верстка, анимации и рабочая форма обратной связи.

## Для рекрутера за 30 секунд

- Формат проекта: промо-страница/персональное портфолио
- Цель: показать качество UI, внимание к деталям и базовую продуктовую подачу
- Стек: HTML, CSS, JavaScript, EmailJS
- Ключевой результат: готовый к публикации лендинг с адаптивом и контактной формой

## Что здесь реализовано

- Чистый адаптивный интерфейс для desktop и mobile
- Секции Home, About, Portfolio, Contact
- Плавная навигация по якорям и анимации появления блоков
- Интерактивные карточки и аккуратные hover-состояния
- Отправка сообщений из формы через EmailJS без собственного бэкенда

## Технический стек

- HTML5
- CSS3 (Grid, Flexbox, custom properties, responsive)
- Vanilla JavaScript
- EmailJS для формы обратной связи

## Быстрый запуск

1. Клонировать репозиторий:

```bash
git clone https://github.com/belyassh/portfolio.git
cd portfolio
```

2. Запустить локально:

```bash
python -m http.server 8000
```

3. Открыть в браузере:

```text
http://localhost:8000
```

## Структура проекта

- [index.html](index.html): разметка страницы
- [styles.css](styles.css): визуальная система, адаптив, анимации
- [script.js](script.js): интерактивность, навигация, логика формы
- [EMAILJS_SETUP.md](EMAILJS_SETUP.md): инструкция по подключению EmailJS

## Настройка формы

Для полноценной отправки сообщений нужно подключить EmailJS.

1. Создать аккаунт на https://www.emailjs.com
2. Получить Public Key, Service ID и Template ID
3. Подставить значения в [script.js](script.js)
4. Подробная инструкция: [EMAILJS_SETUP.md](EMAILJS_SETUP.md)
