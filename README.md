# 🎨 Портфолио Веб-дизайнера

Современный, интерактивный сайт портфолио с красивым дизайном, плавными анимациями и функциональной формой обратной связи.

## ✨ Особенности

- **Современный дизайн** — glass-morphism эффекты и градиентные фоны
- **Адаптивный интерфейс** — полная поддержка мобильных устройств
- **Плавные анимации** — интерактивные элементы и переходы
- **Встроенная форма контактов** — интеграция с EmailJS для отправки сообщений
- **Навигация** — простая и удобная навигация по секциям
- **SEO-оптимизирован** — правильная структура HTML

## 🏗️ Структура проекта

```
portfolio/
├── index.html          # Основная HTML-страница
├── styles.css          # Стили и CSS-анимации
├── script.js           # Логика контактной формы и интерактивность
├── README.md           # Этот файл
└── EMAILJS_SETUP.md    # Инструкция по настройке EmailJS
```

## 🚀 Быстрый старт

### Установка и использование

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/ваш-профиль/portfolio.git
   cd portfolio
   ```

2. **Откройте в браузере**
   - Просто откройте `index.html` в браузере
   - Или используйте локальный сервер:
     ```bash
     python -m http.server 8000
     # затем откройте http://localhost:8000
     ```

3. **Настройте EmailJS** (опционально)
   - Следуйте инструкциям в [EMAILJS_SETUP.md](EMAILJS_SETUP.md)
   - Это нужно, если хотите использовать форму контактов

## ⚙️ Настройка

### Изменение данных

1. **Логотип и название**
   - Отредактируйте HTML в секции Navigation
   - По умолчанию: "hitachi" + "ux"

2. **Текст и информацию**
   - Откройте `index.html`
   - Найдите нужную секцию и отредактируйте текст

3. **Цвета и стили**
   - Отредактируйте `styles.css`
   - Основные переменные находятся в начале файла

### Настройка формы контактов

Полная инструкция находится в файле [EMAILJS_SETUP.md](EMAILJS_SETUP.md).

Вкратце:
1. Создайте аккаунт на [EmailJS](https://www.emailjs.com/)
2. Получите Public Key, Service ID и Template ID
3. Замените значения в `script.js`

## 💡 Как кастомизировать

### Добавьте свои проекты

В секции Portfolio добавьте новые карточки:

```html
<div class="portfolio-item glass-card">
    <div class="portfolio-image">
        <div class="portfolio-overlay">
            <h3>Название проекта</h3>
            <p>Описание</p>
        </div>
    </div>
    <h3>Название проекта</h3>
    <p>Описание проекта</p>
    <a href="#" class="portfolio-link">View Project →</a>
</div>
```

### Измените цветовую схему

В `styles.css` найдите root переменные и отредактируйте цвета:

```css
:root {
    --primary-color: #your-color;
    --secondary-color: #your-color;
    /* и другие переменные */
}
```
