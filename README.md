# UI-UX Landing (hitachiux)

Личный сайт-портфолио для презентации коммерческих кейсов и приема заявок от клиентов.

## Что сделано

- Живой hero-блок с позиционированием под бизнес-задачи
- Сетка реальных кейсов из текущего репозитория
- Превью каждого кейса в модальном окне
- Пакеты услуг с ценовыми диапазонами
- Пошаговый процесс работы
- FAQ-аккордеон
- Форма обратной связи с отправкой через EmailJS
- Безопасный fallback: если EmailJS недоступен, открывается mailto
- Адаптивная мобильная навигация и плавные анимации появления блоков

## Стек

- HTML5
- CSS3
- Vanilla JavaScript
- EmailJS

## Структура

- index.html
- styles.css
- script.js
- EMAILJS_SETUP.md

## Локальный запуск

1. Открой папку проекта в VS Code.
2. Запусти любой локальный сервер, например:

```bash
python -m http.server 8000
```

3. Открой в браузере:

```text
http://localhost:8000
```

## Настройка формы

Смотри подробности в EMAILJS_SETUP.md.

Кратко:

1. Создай сервис и шаблон в EmailJS.
2. Проверь значения в объекте emailConfig в script.js.
3. Протестируй отправку формы.
